# Minimal Themes

Dos temas minimalistas con colores pasteles.

## Funcionalidad

**Temas:**
- **Minimal White** - Tema claro con colores suaves
- **Minimal Black** - Tema oscuro con UI unificado (#1a1a1a)

**Colores Pasteles (ambos temas):**
- Azul: `#5a9fd4`
- Verde: `#6fa872`
- Rosa: `#e08b95`
- Púrpura: `#b87dd4`
- Naranja: `#e8a574`

## Activar Tema

1. `Ctrl/Cmd + K` → `Ctrl/Cmd + T`
2. Selecciona `Minimal White` o `Minimal Black`

**O vía configuración:**

```json
{
  "workbench.colorTheme": "Minimal White"
}
```

## UI Unificado (Minimal Black)

Todos los elementos usan `#1a1a1a`:
- Editor background
- Sidebar
- Panel (terminal, output)
- Title bar
- Tabs
- Inputs
- Status bar
